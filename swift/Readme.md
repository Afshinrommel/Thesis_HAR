The developed iOS application provides a comprehensive user interface designed for real-time inference monitoring, multi-model evaluation, and automated data extraction. The operational flow begins with the selection of one of the four integrated classification models. Once a session is initiated, the system visualizes the real-time probability distributions of motion classes using dynamic bar charts, with the longest bar continuously highlighting the currently dominant activity. Terminating an active session transitions the interface to a summary screen, which reports the top three predicted classes along with their highest confidence scores.

To facilitate comparative analysis, the application includes a concurrent evaluation mode. When this dual-mode functionality is activated, all four models execute simultaneously, providing a consolidated prediction output on the summary screen. Additionally, the system incorporates an automated data export module to support post-deployment analysis. By selecting the "Share Master" function from the main interface, the recorded raw signal logs are formatted into an Excel spreadsheet and automatically transmitted via the device’s default email account.

For researchers intending to use these raw signals with the developed models, it is necessary to compute the total acceleration for each axis by summing the user acceleration and gravity components. Proper axis mapping must also be applied to ensure compatibility with the models.

Finally, a dedicated exit control is provided to safely terminate the application session
